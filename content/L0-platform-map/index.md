
# L0 — Platform Map

Level ini bertujuan membangun pemahaman arsitektural tentang Web Platform.

Level ini tidak membahas API secara detail.
Level ini membahas siapa mendefinisikan apa dan bagaimana layer saling berhubungan.

---

## Tujuan Level Ini

Setelah menyelesaikan L0, Anda harus mampu:

- Membedakan ECMAScript vs Web Platform
- Memahami peran HTML Standard
- Memahami bahwa Event Loop bukan bagian dari JavaScript language
- Memahami bahwa DOM bukan bagian dari ECMAScript
- Memahami bahwa rendering bukan tanggung jawab JavaScript Engine
