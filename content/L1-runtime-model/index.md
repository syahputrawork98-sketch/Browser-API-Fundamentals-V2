# L1 - Runtime Model

Level ini membahas bagaimana browser menjalankan JavaScript dan Web APIs pada runtime.
Level ini adalah fondasi scheduling, rendering, lifecycle, memori, dan concurrency behavior.

---

## Tujuan Level Ini

Setelah menyelesaikan L1, Anda harus mampu:

- Menjelaskan apa itu event loop
- Menjelaskan perbedaan task dan microtask
- Menjelaskan kapan rendering dapat terjadi
- Menjelaskan bagaimana lifecycle dokumen memengaruhi eksekusi
- Menjelaskan dasar retensi memori dan implikasinya
- Menjelaskan model concurrency pada Web Platform
